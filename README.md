# MyBeaconer - iOS iBeacon Development Tool

**MyBeaconer** is a professional iOS application designed for developers and enthusiasts to test, develop, and debug iBeacon transmission and reception functionality.

## Core Capabilities

The platform provides comprehensive iBeacon testing tools. Users can configure and broadcast beacon signals with custom UUID, Major, and Minor values, while simultaneously monitoring nearby beacons with real-time signal strength visualization. QR code integration enables rapid configuration sharing across devices.

## Technical Foundation

Development leverages SwiftUI for the interface layer, with CoreLocation and CoreBluetooth frameworks powering beacon functionality. Realm handles local data persistence for beacon configurations and reception logs.

The system supports contemporary iOS devices running version 17.0 or newer, specifically iPhone models.

## Beacon Management

Users can register up to 10 beacon configurations, each with unique UUID, Major, and Minor identifiers. The transmission interface provides visual feedback with real-time status indicators, while reception monitoring displays proximity detection (Immediate, Near, Far) alongside RSSI measurements.

## Signal Analysis

Beyond basic detection, the app visualizes signal strength through intuitive horizontal bar charts with clearly marked ranges. Reception logs are automatically recorded and organized by date for historical analysis and debugging purposes.

## Configuration Sharing

QR code generation and scanning facilitate rapid beacon configuration exchange between devices. This feature streamlines collaborative development and testing scenarios.

## Additional Details

The application includes both light and dark mode styling with bilingual support (English/Japanese). Comprehensive privacy policy and terms of service documentation ensure transparent data handling practices.

Maintenance and feature development are handled by an individual creator. The app is available on Apple's App Store for developers requiring reliable iBeacon testing tools.

# MyBeaconer - iOS iBeacon開発ツール

**MyBeaconer**は、開発者や愛好家がiBeaconの送受信機能をテスト、開発、デバッグするために設計されたプロフェッショナルなiOSアプリケーションです。

## 主な機能

このプラットフォームは、包括的なiBeaconテストツールを提供します。ユーザーは、カスタムUUID、Major、Minor値を持つビーコン信号を設定・発信でき、同時に近くのビーコンをリアルタイムの信号強度可視化で監視できます。QRコード統合により、デバイス間で素早く設定を共有できます。

## 技術基盤

開発にはSwiftUIをインターフェース層として採用し、CoreLocationとCoreBluetooth フレームワークがビーコン機能を支えています。Realmがビーコン設定と受信ログのローカルデータ永続化を処理します。

このシステムは、iOS 17.0以降を実行する最新のiOSデバイス（iPhone）をサポートしています。

## ビーコン管理

ユーザーは最大10件のビーコン設定を登録でき、それぞれに固有のUUID、Major、Minor識別子を設定できます。送信インターフェースはリアルタイムステータスインジケーターによる視覚的フィードバックを提供し、受信監視はRSSI測定値とともに近接検出（Immediate、Near、Far）を表示します。

## 信号分析

基本的な検出を超えて、アプリは明確にマークされた範囲を持つ直感的な横棒グラフを通じて信号強度を可視化します。受信ログは自動的に記録され、日付ごとに整理されるため、履歴分析とデバッグが容易です。

## 設定共有

QRコード生成とスキャン機能により、デバイス間でビーコン設定を素早く交換できます。この機能は、協調的な開発とテストのシナリオを効率化します。

## その他の詳細

アプリケーションは、ライトモードとダークモードの両方のスタイリングと、バイリンガルサポート（英語/日本語）を備えています。包括的なプライバシーポリシーと利用規約により、透明性のあるデータ取り扱いを保証します。

メンテナンスと機能開発は個人の開発者によって行われています。信頼性の高いiBeaconテストツールを必要とする開発者向けに、Apple App Storeで入手可能です。
